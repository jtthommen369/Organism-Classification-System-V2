# Organism Classification System - Version 2 (V2)

This is **Version 2** of my Organism Classification System, a Python-based interactive program that classifies living and non-living entities based on observable features. 

V2 expands upon the first version by incorporating more organism attributes and distinguishing between animals, plants, fungi/protists, and rocks/minerals.

---

## Features

- Animal classification:
  - Determines whether an organism is **vertebrate or invertebrate**
  - For vertebrates: distinguishes **mammals, birds, reptiles, fish, and amphibians**
  - For invertebrates: identifies **arthropods, mollusks, annelids, and flatworms**
- Plant classification:
  - Classifies as **grass, shrub, or tree**
  - Further identifies **evergreen vs deciduous trees** based on leaf type
- Fungi/Protist identification
- Rocks and minerals classification
- Handles uncertain classifications gracefully

---
